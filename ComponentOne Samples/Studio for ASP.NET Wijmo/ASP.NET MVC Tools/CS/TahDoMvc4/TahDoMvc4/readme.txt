MVC4 To-do List App
-------------------------------------------------------------------------------
A To-do list sample app that fully utilizes the Wijmo MVC Scaffolding.

TahDo is built using the latest in MVC4 technology. It uses the MVC4 Wijmo 
Application Project Template, Entity Framework CodeFirst, Wijmo Widgets and more.
The UI is all generated automatically from the Model using the Wijmo MVC 
Scaffolding. This rich app was created just by defining the Model and
using the Add Controller dialog.

This sample demonstrates how you can generate CRUD applications quickly and 
easily from a well-defined model.
