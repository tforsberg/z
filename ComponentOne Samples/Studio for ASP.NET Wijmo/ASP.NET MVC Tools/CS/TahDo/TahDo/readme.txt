MVC To-do List App
-------------------------------------------------------------------------------
A To-do list sample app that fully utilizes the Wijmo MVC Scaffolding.

TahDo is built using the latest in MVC3 technology. It uses the MVC3 Wijmo 
Application Project Template, Entity Framework CodeFirst, jQuery and more.
The UI is all generated automatically from the Model using the Wijmo MVC 
Scaffolding. This rich app was created just by defining the Model and
using the Add Controller dialog.
