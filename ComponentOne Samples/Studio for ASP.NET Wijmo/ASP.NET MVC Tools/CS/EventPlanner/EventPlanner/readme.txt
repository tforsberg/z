MVC EventPlanner App
-------------------------------------------------------------------------------
An EventPlanner sample app that fully utilizes the Wijmo Mobile MVC Scaffolding.

EventPlanner is built using the latest in MVC4 technology and AppView widget. 
It uses the MVC4 Wijmo Mobile Application Project Template, Entity Framework CodeFirst, jQuery and more.
The UI is all generated automatically by AppView widget and by Wijmo MVC Scaffolding from the Model.
