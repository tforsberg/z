ListBox

The ListBox control displays a list of items that can contain plain text or HTML, and allows users to select items with the mouse or keyboard.

These samples use the WjListBox directive and AngularJS.

You can use an array of strings as the source of items for the ListBox using the itemsSource property:

Selected Item: {{ctx.listBox1.selectedItem}}
Selected Index: {{ctx.listBox1.selectedIndex}}


You can also use an array of objects in the itemsSource, provided you also set the displayMemberPath property to indicate which property of the objects to display in the list:

Selected Item: {{ctx.listBox2.selectedItem.country}}
Selected Index: {{ctx.listBox2.selectedIndex}}


You can use HTML in the ListBox control by setting the isContentHtml property to true:

Selected Item: {{ctx.listBox3.selectedItem}}
Selected Index: {{ctx.listBox3.selectedIndex}}
