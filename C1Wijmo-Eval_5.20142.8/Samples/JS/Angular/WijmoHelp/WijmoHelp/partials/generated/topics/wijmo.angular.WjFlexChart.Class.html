AngularJS directive for the FlexChart control.

Use the WjFlexChart directive to add charts to your AngularJS applications. Note that directive and parameter names must be formatted as lower-case with dashes instead of camel-case. For example:

Here is a <b>FlexChart</b> control:</p>
<wj-flex-chart style="height:300px" items-source="data" binding-x="country">
  <wj-flex-chart-axis axis-type="Y" major-unit="5000"></wj-flex-chart-axis>
  <wj-flex-chart-series binding="sales" name="Sales"></wj-flex-chart-series>
  <wj-flex-chart-series binding="expenses" name="Expenses"></wj-flex-chart-series>
  <wj-flex-chart-series binding="downloads" name="Downloads" chart-type="LineSymbols"></wj-flex-chart-series>
</wj-flex-chart>

The example below creates a FlexChart control and binds it to a 'data' array exposed by the controller. The chart has three series, each corresponding to a property in the objects contained in the source array. The last series in the example uses the 'chart-type' attribute to override the default chart type used by the other series.

The WjFlexChart directive supports the following attributes:

binding
@ Name of the property that contains the X values (may be overridden by series).
bindingX
@ Name of the property that contains the Y values (may be overridden by series).
chartType
@ Default chart type (may be overridden by each series, see ChartType).
control
= Reference to the FlexChart control created by this directive.
footer
@ Text displayed in the chart footer (plain text).
footerStyle
= Style applied to the chart footer.
header
@ Text displayed in the chart header (plain text).
headerStyle
= Style applied to the chart header.
interpolateNulls
@ Whether to interpolate or leave gaps when there are null values in the data.
itemFormatter
= Formatter function.
itemsSource
= Array or ICollectionView that contains the data used to create the chart.
legendToggle
@ Whether clicking the legend items should toggle series visibility.
options
= Miscellaneous chart options.
palette
= Array with default colors used for displaying each series.
plotMargin
= Plot margin.
rotated
@ Whether to flip the axes so X is vertical and Y horizontal.
selection
= Chart selection.
selectionMode
@ Chart selection mode (see SelectionMode).
stacking
@ Whether series should be stacked or plotted independently (see Stacking).
symbolSize
@ Size of the symbols used to render this FlexChart.
rendered
& The rendered event handler (see rendered).
seriesVisibilityChanged
& The seriesVisibilityChanged event handler (see seriesVisibilityChanged).
selectionChanged
& The selectionChanged event handler (see selectionChanged).

The WjFlexChart directive may contain the following child directives: WjFlexChartAxis and WjFlexChartSeries.
