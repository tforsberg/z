CellRange Class

Module
wijmo.grid
File
wijmo.grid.js

Represents a rectangular group of cells defined by two row indices and two column indices.


Constructor

Name Parameters Description
constructor(r, c, r2, c2)
  • r (number): Index of the first row in this range. (default Value: -1)
  • c (number): Index of the first column in this range. (default Value: -1)
  • r2 (number): Index of the last row in this range. (default Value: r)
  • c2 (number): Index of the first column in this range. (default Value: c)
 Initializes a new instance of a CellRange.
