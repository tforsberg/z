AngularJS directive for the FlexGrid Column object.

The WjFlexGridColumn directive must be contained in a WjFlexGrid directive. It supports the following attributes:

aggregate
@ Aggregate to display in the group header rows for this column.
align
@ Horizontal alignment of items in this column.
allowDragging
@ Whether the user can move this column to a new position with the mouse.
allowSorting
@ Whether the user can sort this column by clicking its header.
allowResizing
@ Whether the user can resize this column with the mouse.
allowMerging
@ Whether cells in this column can be merged.
binding
@ Name of the property this column is bound to.
cssClass
@ Name of a CSS class to be used when rendering this column.
dataMap
= DataMap used to convert raw values into display values for this column.
dataType
@ Type of value stored in this column (see DataType).
format
@ Format string used to convert raw values into display values for this column (see Globalize).
header
@ String displayed in the column header.
isContentHtml
@ Whether cells in this column contain HTML content rather than plain text.
isReadOnly
@ Whether the user can edit values in this column.
isSelected
@ Whether this column is selected.
maxWidth
@ Maximum width for this column.
minWidth
@ Minimum width for this column.
name
@ Column name (can be used to retrieve the column).
visible
@ Whether this column is visible.
width
@ Width of this column (in pixels or as a star value).
wordWrap
@ Whether cells in this column should wrap their content.

In addition to regular attributes that match properties in the Column class, WjFlexGridColumn directives may contain an ng-style attribute that provides conditional formatting and an HTML fragment that is used as a cell template.

Both the ng-style attribute and the HTML fragment can use the $item property in AngularJS bindings to refer to the item that is bound to the current row. For example:

<wj-flex-grid-column header="Symbol" binding="symbol" read-only="true" width="*">
  <a href="https://finance.yahoo.com/q?s={​{$item.symbol}}">
    {​{$item.symbol}}
  </a>
</wj-flex-grid-column>
<wj-flex-grid-column header="Change" binding="changePercent" format="p2" width="*"
  ng-style="{color:getAmountColor($item.change)}">
</wj-flex-grid-column>

These directives create two columns. The first has a template that produces a hyperlink based on the bound item's "symbol" property. The second has a conditional style that renders values with a color determined by a function implemented in the controller.
