Merging

All versions of the FlexGrid support content-driven cell merging.

To enable cell merging, set the allowMerging property to indicate what part(s) of the grid you want to merge, and set the allowMerging property on specific rows and columns to true. Once you do that, the grid will automatically merge cells that have the same content, grouping the data visually.

In the example below, the Country, Product, Color, and Active columns have their allowMerging property set to true.

Try changing the value of the grid's allowMerging property and sorting the merged columns to see merging in action.
