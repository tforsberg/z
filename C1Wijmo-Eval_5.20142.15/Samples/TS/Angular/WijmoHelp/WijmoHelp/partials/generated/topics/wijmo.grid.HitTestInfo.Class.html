HitTestInfo Class

Module
wijmo.grid
File
wijmo.grid.js

Contains information about a part of a FlexGrid control at a specified page coordinate.


Constructor

Name Parameters Description
constructor(grid, pt)
  • grid (any): FlexGrid control or GridPanel to investigate.
  • pt (any): Point in page coordinates to investigate.
 Initializes a new instance of a HitTestInfo.

Properties

Name Type Description
cellRange CellRange Gets the cell range at the specified position.
cellType CellType Gets the cell type at the specified position.
col number Gets the column index of the cell at the specified position.
edgeBottom boolean Gets whether the mouse is near the bottom edge of the cell.
edgeLeft boolean Gets whether the mouse is near the left edge of the cell.
edgeRight boolean Gets whether the mouse is near the right edge of the cell.
edgeTop boolean Gets whether the mouse is near the top edge of the cell.
gridPanel GridPanel Gets the grid panel at the specified position.
point Point Gets the point in control coordinates that this HitTestInfo refers to.
row number Gets the row index of the cell at the specified position.
