AngularJS directive for the FlexChart Axis object.

The WjFlexChartAxis directive must be contained in a WjFlexChart directive. It supports the following attributes:

wjProperty
@ Defines the FlexChart property name, axisX or axisY, to initialize with this directive.
axisLine
@ Whether the axis line is visible.
format
@ Format string used for the axis labels (see Globalize).
labels
@ Whether the axis labels are visible.
majorGrid
@ Whether the axis should include grid lines.
majorTickMarks
@ Defines the appearance of the tick marks (see TickMark).
majorUnit
@ Distance between the labels and the.
max
@ Minimum value shown on the axis.
min
@ Maximum value shown on the axis.
position
@ Axis position (see Position).
reversed
@ Whether the axis is reversed (top to bottom/right to left).
title
@ Text shown next to the axis.
