IEditableCollectionView Interface

Implements
ICollectionView
Module
wijmo.collections
File
wijmo.js

Defines methods and properties that extend ICollectionView to provide editing capabilities.


Properties

Name Type Description
F canAddNew boolean Gets a value that indicates whether a new item can be added to the collection.
F canCancelEdit boolean Gets a value that indicates whether the collection view can discard pending changes and restore the original values of an edited object.
F canRemove boolean Gets a value that indicates whether items can be removed from the collection.
F currentAddItem any Gets the item that is being added during the current add transaction.
F currentEditItem any Gets the item that is being edited during the current edit transaction.
F isAddingNew boolean Gets a value that indicates whether an add transaction is in progress.
F isEditingItem boolean Gets a value that indicates whether an edit transaction is in progress.

Methods

Name Parameters Return Type Description
addNew() any Adds a new item to the collection.
Returns The new item that was added to the collection..
cancelEdit() void Ends the current edit transaction and, if possible, restores the original value to the item.
cancelNew() void Ends the current add transaction and discards the pending new item.
commitEdit() void Ends the current edit transaction and saves the pending changes.
commitNew() void Ends the current add transaction and saves the pending new item.
editItem(item)
  • item (any): Item to edit.
 void Begins an edit transaction of the specified item.
remove(item)
  • item (any): Item to remove from the collection.
 void Removes the specified item from the collection.
removeAt(index)
  • index (number): Index of the item to remove from the collection.
 void Removes the item at the specified index from the collection.
