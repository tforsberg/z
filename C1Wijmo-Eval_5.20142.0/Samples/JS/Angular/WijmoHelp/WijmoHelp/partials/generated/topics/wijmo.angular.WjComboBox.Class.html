AngularJS directive for the ComboBox control.

Use the WjComboBox directive to add ComboBox controls to your AngularJS applications. Note that directive and parameter names must be formatted as lower-case with dashes instead of camel-case. For example:

<p>Here is a ComboBox control:</p>
<wj-combo-box 
  text="theCountry" 
  items-source="countries"
  is-editable="false" 
  placeholder="country">
</wj-combo-box>

The example below creates a ComboBox control and binds it to a 'countries' array exposed by the controller. The ComboBox searches for the country as the user types. The isEditable property is set to false, so the user is forced to select one of the items in the list.

The WjComboBox directive supports the following attributes:

control
= Reference to the ComboBox control created by this directive.
displayMemberPath
@ Name of the property to use as the visual representation of the items.
isContentHtml
@ Whether the drop-down list should display the items as plain text or as HTML.
isDroppedDown
@ Whether the drop down is currently visible.
isEditable
@ Whether the user should be allowed to enter values not present on the list.
itemFormatter
= Function used to customize the values shown on the drop-down list.
itemsSource
= Array or ICollectionView that contains items shown on the list.
maxDropDownHeight
@ Maximum height of the drop-down list.
maxDropDownWidth
@ Maximum width of the drop-down list.
placeholder
@ String shown as a hint when the control is empty.
showDropDownButton
@ Whether the control should display a drop-down button.
selectedIndex
= Index of the currently selected item in the drop-down list.
selectedItem
= Item that is currently selected item in the drop-down list.
selectedValue
= Value of the selected item, obtained using the selectedValuePath.
selectedValuePath
@ Name of the property used to get the selectedValue from the selectedItem.
text
= Text shown in the control.
isDroppedDownChanged
& The isDroppedDownChanged event handler (see isDroppedDownChanged).
selectedIndexChanged
& The selectedIndexChanged event handler (see selectedIndexChanged).
textChanged
& The textChanged event handler (see textChanged).
