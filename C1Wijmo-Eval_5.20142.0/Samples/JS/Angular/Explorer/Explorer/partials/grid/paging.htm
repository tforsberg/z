Paging

This view shows how to implement paged views with the FlexGrid. All the work is done by the CollectionView class used as a data source for the grid. To add paging, set the CollectionView.pageSize property, exactly as you would do in .NET. To switch pages, use the CollectionView.moveToPage method.

Note that the paging UI is implemented outside the grid. This gives the page author complete control over the appearance and functionality of the paging mechanism. For example, in this sample the paging UI is implemented with Angular directives and support switching pages as well as page size.

Note also that the grid element has its max-height set to 400 pixels. If the page size is small, the grid grows to accommodate its contents. If the page size is large, the grid height grows to 400 pixels and a vertical scroll bar is automatically displayed.
