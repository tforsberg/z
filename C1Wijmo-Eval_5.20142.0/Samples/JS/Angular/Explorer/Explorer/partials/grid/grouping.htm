Grouping

This view shows grouping support in the FlexGrid. Most of the work is done by the CollectionView class used as a data source for the grid. To add grouping, add one or more GroupDescription objects to the CollectionView.groupDescriptions property, exactly as you would do in .NET, and make sure the grid's showGroups property is set to true.

You may customize the text that is displayed in group header rows using the groupHeaderFormat property.
