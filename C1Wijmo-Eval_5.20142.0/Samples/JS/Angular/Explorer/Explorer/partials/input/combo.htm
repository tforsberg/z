ComboBox

The ComboBox control is a simple and powerful combo box. These are the main properties of the ComboBox control:

itemsSource
Gets or sets an array or an ICollectionView with the options available.
displayMember
If the items are objects (as opposed to strings), use this property to specify which property of the items should be displayed in the ComboBox.
text
Gets or sets the text in the control.
selectedIndex
Gets or sets the index of the currently selected item in the itemsSource.
selectedItem
Gets or sets currently selected item in the itemsSource.
isEditable
Gets or sets whether the user can type values that are not in the itemsSource.

The itemsSource and displayMember properties are responsible for populating the option list. They are mainly input properties. The text, selectedItem, and selectedIndex properties change to reflect the user input. They are mainly output properties.
