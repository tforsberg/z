ListBox

The ListBox control displays a list of items, which may contain plain text or HTML, and allows users to select items with the mouse or keyboard.

You may use an array of strings as the items source for the ListBox:

Selected Item: {{ctx.listBox1.selectedItem}}
Selected Index: {{ctx.listBox1.selectedIndex}}


Or you may use an array of objects and specify the displayMemberPath property to indicate which property of the objects should be displayed in the list:

Selected Item: {{ctx.listBox2.selectedItem.country}}
Selected Index: {{ctx.listBox2.selectedIndex}}


And you can use HTML in your ListBox controls by setting the isContentHtml property to true:

Selected Item: {{ctx.listBox3.selectedItem}}
Selected Index: {{ctx.listBox3.selectedIndex}}
