AngularJS directive for the FlexChart Series object.

The WjFlexChartSeries directive must be contained in a WjFlexChart directive. It supports the following attributes:

binding
@ Name of the property that contains the X values.
bindingX
@ Name of the property that contains the Y values.
chartType
@ Chart type (see ChartType).
cssClass
@ The series CSS class.
itemsSource
= Array or ICollectionView that contains the data for this series.
name
@ Name of the series as shown in the legend.
style
= The series style.
symbolMarker
@ The series marker. (see Marker).
symbolSize
@ Size of the symbols used to render this Series.
symbolStyle
= The series symbol style.
visibility
= The series visibility. (see SeriesVisibility).

In most cases, the WjFlexChartSeries specifies only the "name" and "binding" properties. The remaining values are inherited from the parent WjFlexChart directive.
