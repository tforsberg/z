Axis Class

Implements
_IAxis
Module
wijmo.chart
File
wijmo.chart.js

Represents an axis in the chart.


Constructor

Name Parameters Description
constructor(position)
  • position (Position): Position of the axis on the chart.
 Initializes a new instance of an Axis.

Properties

Name Type Description
actualMax number Gets the actual axis maximum.
actualMin number Gets the actual axis minimum.
axisLine boolean Gets or sets whether the axis line is visible.
axisType AxisType Gets the axis type.
format string Gets or sets the format string used for the axis labels (see Globalize).
labels boolean Gets or sets whether the axis labels are visible.
majorGrid boolean Gets or sets whether the axis should include grid lines.
majorTickMarks TickMark Gets or sets the appearance of axis tick marks.
majorUnit number Gets or sets the distance between axis labels.
max number Gets or sets the maximum value shown on the axis. If not set, the maximum is calculated automatically.
min number Gets or sets the minimum value shown on the axis. If not set, the minimum is calculated automatically.
position Position Gets or sets the axis position.
reversed boolean Gets or sets whether the axis is reversed (top to bottom/right to left).
title string Gets or sets the text shown next to the axis.
