AngularJS directive for the ListBox control.

Use the WjListBox directive to add ListBox controls to your AngularJS applications. Note that directive and parameter names must be formatted as lower-case with dashes instead of camel-case. For example:

b>Here is a ListBox control:</p>
<wj-list-box
  selected-item="theCountry" 
  items-source="countries"
  placeholder="country">
</wj-list-box>

The example below creates a ListBox control and binds it to a 'countries' array exposed by the controller. The value selected is bound to the 'theCountry' controller property using the selectedItem attribute.

The WjListBox directive supports the following attributes:

control
= Reference to the ListBox control created by this directive.
displayMemberPath
@ Property to use as the visual representation of the items.
isContentHtml
@ Whether items contain plain text or HTML.
itemFormatter
= Function used to customize the values shown on the list.
itemsSource
= Array or ICollectionView that contains the list items.
maxHeight
@ The maximum height of the list.
selectedIndex
= Index of the currently selected item.
selectedItem
= Item that is currently selected.
selectedValue
= Value of the selectedItem obtained using the selectedValuePath.
selectedValuePath
@ Property used to get the selectedValue from the selectedItem.
itemsChanged
& The itemsChanged event handler (see itemsChanged).
selectedIndexChanged
& The selectedIndexChanged event handler (see selectedIndexChanged).
