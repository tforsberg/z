RowColFlags Enum

Module
wijmo.grid
File
wijmo.grid.js

Flags that specify the state of a grid row or column.


Values

Name Description
AllowDragging Row/Column can be dragged to a new position with the mouse.
AllowMerging Row/Column can contain merged cells.
AllowResizing Row/Column can be resized.
AllowSorting Column can be sorted by clicking its header with the mouse.
AutoGenerated Column was generated automatically.
Collapsed Group row is collapsed.
ColumnDefault Default settings for new columns.
HtmlContent Cells in this row/column contain HTML text.
ParentCollapsed Row has a parent group in collapsed state.
ReadOnly Row/Column is read-only (cannot be edited).
RowDefault Default settings for new rows.
Selected Row/Column is selected.
Visible Row/Column is visible.
WordWrap Cells in this row/column may contain wrapped text.
